General Structure
------------

Our code structure in the most recent year is strongly based on WPILib's hatchbottraditional example project. To see the structure of this project, create a new example project in WPILib VSCode and select Hatch Bot (Traditional) as the example project type.

Some key points about our structure:
   - 1 Constants file
   - 1 Robot Container file
   - Subsystem directory
   - Other stuffs
