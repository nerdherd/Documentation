General Structure
------------

Our code structure in the most recent year is strongly based on WPILib's hatchbottraditional example project. To see the structure of this project, create a new example project in WPILib VSCode and select Hatch Bot (Traditional) as the example project type. Also, try to keep your code as clean and elegant as possible, and make sure to make useful comments.
